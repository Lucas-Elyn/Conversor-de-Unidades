## Conversor de Unidades Simples (Android)

Este projeto consiste em um aplicativo Android simples para conversão de unidades de medida. Desenvolvido utilizando **Kotlin** e a biblioteca de interface do usuário moderna **Jetpack Compose**, o conversor oferece uma maneira intuitiva de transformar valores entre diferentes unidades de comprimento. Atualmente, o aplicativo suporta a conversão entre **Centímetros, Metros, Feet e Milímetros**.

### Funcionalidades Principais

* **Interface Intuitiva:** Interface de usuário declarativa e reativa construída com Jetpack Compose.
* **Seleção Dinâmica de Unidades:** Menus dropdown permitem escolher as unidades de entrada e saída desejadas.
* **Conversão Instantânea:** O resultado da conversão é exibido em tempo real à medida que as unidades são selecionadas ou o valor de entrada é modificado.
* **Suporte a Unidades Comuns de Comprimento:** Inclui suporte para as unidades Centímetros, Metros, Feet e Milímetros.

### Como Usar

1.  **Insira o Valor:** Digite o valor numérico que você deseja converter no campo de texto "Enter Value:".
2.  **Selecione a Unidade de Entrada:** Toque no botão exibindo a unidade de entrada atual e escolha a desejada no menu suspenso.
3.  **Selecione a Unidade de Saída:** Toque no botão exibindo a unidade de saída atual e escolha a unidade para a qual você quer converter o valor.
4.  **Visualize o Resultado:** O valor convertido será mostrado automaticamente abaixo dos seletores de unidade no formato "Result: \[valor convertido] \[unidade de saída]".
